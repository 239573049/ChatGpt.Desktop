# ChatGpt.Desktop

## 介绍
一个基于avalonia实现的仿webChat的ChatGpt客户端
支持聊天记录，支持聊天连续

## 软件架构
使用`Avalonia`作为跨平台支持 `.NET 7`，界面仿制`WebChat`

## 使用说明

1.  打开设置 => 设置ChatGpt的token
2.  保存设置
3.  输入需要提问的问题
4.  得到答案
5.  点击答案可直接复制内容
